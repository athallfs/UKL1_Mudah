# Program ini digunakan untuk menghitung total biaya pengiriman paket Menggunakan bahasa java berdasarkan
-Berat paket (dalam kilogram)

-Jarak pengiriman (dalam kilometer)

-Volume paket (dalam cm³, dihitung dari panjang × lebar × tinggi)

Biaya pengiriman ditentukan oleh:

-Jika jarak ≤ 10 km, maka biaya per kilogram = Rp 4.250

-Jika jarak > 10 km, maka biaya per kilogram = Rp 6.000

-Jika volume > 100 cm³, maka ditambah biaya Rp 50.000
